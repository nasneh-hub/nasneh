# Dashboard

Unified dashboard for Admin, Vendor, Provider, and Driver roles.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI

## Routes

- `/admin/*` - Admin dashboard
- `/vendor/*` - Vendor dashboard
- `/provider/*` - Service Provider dashboard
- `/driver/*` - Driver dashboard

## Development

```bash
pnpm dev
```

## URL

Production: https://dashboard.nasneh.com
