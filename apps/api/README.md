# Nasneh API

Backend API for Nasneh marketplace.

## Tech Stack

- Node.js
- Express
- Prisma ORM
- PostgreSQL
- Redis (cache)
- JWT Authentication

## Development

```bash
pnpm dev
```

## URL

Production: https://api.nasneh.com
